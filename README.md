# Endorsement Operations Labs — Claude Plugin Marketplace

A Claude plugin marketplace for the endorsement operations training labs.

## Add it in Claude Desktop
Settings → Plugins → Add marketplace → enter `courselabfiles/endorsement-ops-marketplace` → Sync.
Then install the `endorsements-handoff` and/or `endorsement-sla-analytics` plugins.

## Contents
- `endorsements-handoff` — weekly SI-Queue handoff: case view, Excel export, draft email.
- `endorsement-sla-analytics` — cycle-time and SLA analytics with $-at-risk defect ranking.

The plugins call the Salesforce Demo and Jira Demo connectors, installed separately as Claude Desktop extensions.
